# Spring Boot 3 Todo Application  
**Backend:**
* Spring Boot 3.0.2
* H2 Database
* Spring Data JPA
  
**Frontend:**
* Bootstrap
* Thymeleaf
## How to use
Run in terminal:
* `git clone https://github.com/alextoritsin/spring-todolistapp.git`  
* `cd spring-todolistapp`
* `./mvnw spring-boot:run`    

Go to http://localhost:8080 and add your todo's.
